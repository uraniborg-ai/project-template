# Uraniborg Project Template

This is a uv-based Python project template for Uraniborg AI research and
AI-assisted coding workflows.

## Quick Start

```sh
scripts/bootstrap.sh
```

For manual setup, dependency management, and formatting commands, see
[docs/uv.md](docs/uv.md).

## Standard Commands

```sh
uv sync --all-groups
uv run python --version
uv run ruff format .
uv run ruff check .
```
